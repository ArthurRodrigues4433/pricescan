Termos de Uso — PriceScan

1. Sobre o serviço
O PriceScan é um aplicativo de auxílio ao controle de compras de feira. Ele utiliza OCR (reconhecimento óptico de caracteres) para extrair dados de fotos de cartazes de preços.

2. Limitações do OCR
A leitura automática de cartazes pode conter erros. O usuário é responsável por conferir e corrigir todos os dados antes de confirmá-los. O sistema exibe os dados lidos apenas como sugestão inicial.

3. Responsabilidade pelos dados
Ao confirmar um produto (etapa "Conferir dados do cartaz"), o usuário declara que verificou as informações e que elas estão corretas. O PriceScan não se responsabiliza por decisões de compra baseadas em dados não conferidos pelo usuário.

4. Dados pessoais
Apenas o e-mail e senha são armazenados para autenticação. Nenhum dado é compartilhado com terceiros. As listas de compra ficam armazenadas apenas na conta do usuário.

5. Uso adequado
O aplicativo destina-se a uso pessoal. É vedado o uso automatizado, scraping ou qualquer forma de abuso da plataforma.

6. Alterações
Estes termos podem ser atualizados. Alterações relevantes serão comunicadas no próprio aplicativo.
